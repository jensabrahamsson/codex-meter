name: usage-dashboard
description: Show local Codex token usage and recent activity.
---

Use Codexmeter when the user asks about token usage, recent activity, or rolling
windows such as the last minute, last hour, or last day.

Prefer the local dashboard at `http://127.0.0.1:8080` for visual inspection.
Use the MCP tools when the user wants a direct numeric answer.

Report:
- `1m`, `1h`, and `24h` token totals
- input, cached input, output, and reasoning token splits
- the most recent active thread identifiers when useful
